# Feluca Rooms – Documentazione Operativa

Questo repository contiene tutta la documentazione ufficiale di Feluca Rooms, organizzata in modo chiaro e scalabile.  
L’obiettivo è creare un sistema operativo completo, standardizzato e facilmente delegabile.

## 📂 Contenuti principali

- **SOP (Standard Operating Procedures)**
  - Check-in
  - Check-out
  - Manutenzione
  - Pulizie
  - Onboarding inquilino
  - Gestione emergenze

- **Policy operative**
  - Regole interne
  - Standard di comunicazione
  - Linee guida per la qualità

- **Template**
  - Messaggi WhatsApp
  - Email
  - Comunicazioni standard

## 🎯 Obiettivo del repository

Costruire un sistema operativo completo per Feluca Rooms che permetta:
- scalabilità
- delega
- automazione
- chiarezza operativa
- riduzione degli errori

## 🧭 Struttura dei file

Ogni SOP è salvata in formato Markdown (`.md`) per garantire:
- leggibilità
- versionamento
- compatibilità con GitHub e Copilot

## 🤖 Supporto Copilot

Questo repository è ottimizzato per lavorare con GitHub Copilot Pro:
- generazione automatica di SOP
- miglioramento dei processi
- standardizzazione dei documenti
- suggerimenti intelligenti

---

Per qualsiasi modifica o nuova SOP, creare un nuovo file `.md` e mantenere la struttura standard.
